## Getting Started

To run in local:
#### 1. Go to `next.config.js` and change this value to `false`.
```
  hosted: false
```

#### 2. Next run these commands to start the project.
```bash
yarn install
yarn dev
```

#### 3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
